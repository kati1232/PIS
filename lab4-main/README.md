# Лабораторная работа №4: Создание страницы определенной записи

## Описание проекта
Проект блога с системой статей. Реализованы:
- Главная страница со списком всех статей (архив)
- Страница отдельной статьи с полным текстом
- Стилизация в соответствии с макетом

## Особенности реализации:
1. Использование регулярных выражений в URL
2. Обработка исключений (404 при отсутствии статьи)
3. Ссылки между страницами (заголовки → статья, "Все записи" → архив)
